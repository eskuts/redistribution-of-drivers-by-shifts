# Распределение водителей по сменам

## описание
    Оптимальное распределения смен с учётом минимальной стоимости рейса
    и знания водителем маршрута. С использованием матрицы затрат и мат. моделирования

## пример запуска
    ```python main.py --coef_period "01.04.2024-30.06.2024" --month_number 6```
